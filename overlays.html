Tooltips

  • Tooltip (Default Settings) - Default settings (width/direction).
  • Tooltip (Set Direction) - Default width and a set direction (bottom).
  • Tooltip (Set Width) - 100px width and default direction.
  • Tooltip (Settings Applied) - 100px width and a set direction (top).
  • Tooltip (Settings Applied) - Same settings as above, but the content of the tooltip is taken from title attribute. The HTML structure is being created.
    This should be used if you want a tooltip with minimal text and formatting (i.e. using headers, bold text, etc...).
